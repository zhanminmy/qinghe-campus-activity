青禾校园活动站｜软件工程实践作业
================================

一、运行方法
1. 解压提交包。
2. 双击 index.html，即可在浏览器中打开网站。
3. 建议使用 Chrome、Edge、Safari 等现代浏览器。

二、网站页面
- index.html：首页，包含轮播、精选活动、数字动画等内容。
- activities.html：活动中心，支持按类别筛选。
- detail.html：活动详情，支持活动数据切换、报名弹窗、表单校验和剩余名额更新。
- about.html：关于我们，包含留言表单和校验反馈。

三、代码目录
- css/common.css：全站布局、组件、动画和响应式样式。
- js/activities-data.js：六项活动的业务数据。
- js/main.js：全站交互逻辑与页面渲染。
- HTML 文件：页面语义结构与具体内容。

四、主要功能
- 自动轮播及手动切换
- 页面滚动渐入
- 数据增长动画
- 活动分类筛选
- 深色/浅色主题切换
- 移动端导航
- 报名弹窗与表单验证
- 报名成功后更新剩余名额（数据保存在当前浏览器中）
- 留言表单验证
- 返回顶部按钮
- 响应式页面布局

五、部署为公开网址（可选）
可将整个 campus-activity 文件夹上传到 GitHub Pages、Vercel 或学校提供的
静态网站空间。首页文件必须保持名称为 index.html。

六、提交前检查
[ ] 双击 index.html，逐个点击导航和功能，确认显示正常。
[ ] 按学校平台要求上传 ZIP 文件，不要只上传某一个 HTML 页面。

说明：本项目为纯前端演示作品，不收集或上传真实个人信息。
